=head2 EBT stats tool

NoteInfoGenerator is (apparently) dead. Here's my own attempt at it.

=head2 Deployment under apache mod_proxy

=over 4

=item Apache configuration

 Alias /ebtdev/ /some/path/
 ProxyRequests Off
 ProxyPreserveHost On
 ProxyPass /ebtdev/ebtst/stats !
 ProxyPass /ebtdev/ebtst http://localhost:1234/ebtdev/ebtst keepalive=On
 ProxyPassReverse /ebtdev/ebtst http://localhost:1234/ebtdev/ebtst
 ProxyTimeout 2500

=item EBTST configuration

 base_href = http://example.com/ebtdev/ebtst/
 statics_dir = /some/path/ebtst/public
 hypnotoad_listen = http://localhost:1234
 hypnotoad_is_proxy = 1
 hypnotoad_heartbeat_timeout = 2500

=over 4

=item Chart::Gnuplot (and gnuplot, of course)

=back

=back
